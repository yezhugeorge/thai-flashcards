# 活人泰语点读卡 - 项目记忆

## 项目概述
- **名称**: 活人泰语点读卡 🇹🇭
- **类型**: 纯前端 Web 应用（HTML + CSS + JS，无构建步骤）
- **用途**: 泰国生活实用泰语学习，点读卡片式
- **原部署链接**: https://68a3b6628ce343a88c0fd0add8133db5.app.codebuddy.work

## 文件结构
- `index.html` - 主页面（SPA 结构）
- `css/styles.css` - 样式（~21KB）
- `js/data.js` - 泰语数据（~50KB，14个分类，含句子/读音/翻译）
- `js/app.js` - 应用逻辑（~25KB，路由/搜索/收藏/音频播放）

## 功能
- 14 个生活场景分类（租房、打车、餐厅、7-11、按摩、看病、恋爱、交友、购物等）
- 100 句系列（日常/购物/餐饮/出行/社交各 20 句）
- 搜索功能、收藏功能、难度/语气筛选
- 泰语发音播放
- 移动端优先设计，底部导航

## 数据来源
《来泰国后真的能用上的活人泰语》PDF，语气词统一为女性用语（ค่ะ/คะ）

## 用户需求
- 用户希望长期保存这个应用（至少一年），不希望链接失效
